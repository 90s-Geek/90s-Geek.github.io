Week 2

In week 2 of WEBD-225 we focused on using CSS and preprocessors to style web pages. One thing I took away was the use of Sass variables, which use the "$" prefix. This allows you to define a value like a color once and reuse it throughout your stylesheet, making updates much more efficient. I also learned about deploying a live page through Github.
